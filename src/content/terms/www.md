## 初級

ブラウザで見るウェブ全体を指す呼び方。

**World Wide Web** の略で、リンクでつながったページの集合のイメージです。古い文書にもよく残ります。

## 中級

日常の「Web サイト」と混ざりやすく、**サイト全体**なのか **URL の `www.` 部分**なのか文脈が曖昧になりがちです。正規化や DNS の話につながります。

URL の `www.` サブドメイン慣習と、DNS の CNAME / 正規化（カノニカル）の話がセットになります。

## 上級

HTTP の Host ヘッダ、クッキーのスコープ、HSTS とサブドメイン戦略が統合設計の論点になります。
